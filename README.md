# Hangman (Java)

A classic Hangman game built in Java. This project was created to practice core Java programming concepts such as loops, conditionals, arrays or collections, string manipulation, and user input handling.

## Features

- Single-player Hangman game
- Random word selection
- Letter-by-letter guessing
- Limited number of incorrect guesses
- Win and lose conditions
- Text-based gameplay
- Written entirely in Java

## Technologies Used

- Java (JDK 8 or higher recommended)
- Java Standard Library

## How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/hangman-java.git

2. Navigate the project directory
  cd hangman-java

3. Compile the java files
   javac *.java

4. Run the game
   java control
